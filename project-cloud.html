Cloud Computing

This project is on "Privacy Preserving Ranked Multi-Keyword Search for Multiple Data Owners in Cloud Computing", which is based on Encryption and Decryption technique, used for files and sensitive data and documents, over the cloud.


This website has been developed for security concerns. As security is the main thing that we always want while accessing files over internet. So that only authorized users can access the files. This could also help any organization keep their confidential files safe (as the files will be encrypted), and only those people can access the files those who have access to it and thus the files are decrypted to only those users.


The front-end is developed using HTML and CSS and Back-end is developed using Java and J2EE technology and for its backend MySQL will be used.
It basically uses the cloud computing as its domain.
